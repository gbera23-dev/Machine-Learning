პირველი დავალება არის kaggle - ის competition House Prices - Advanced Regression Techniques.

შესრულების ინსტრუქცია:

ამ დავალებაში თქვენი ამოცანაა Cleaning, Feature Engineering და Feature Selection - ის სხვადასხვა მიდგომების გატესტვა competition - ის მონაცემებზე, მოდელის სხვადასხვა ჰიპერპარამეტრების გატესტვა და მათი შედეგების სწორი ანალიზი და ყველა ჩატარებული ექსპერიმენტის შესაბამისი ინფორმაციის MLflow ზე დალოგვა.

ამ დავალებისათვის უნდა შექმნათ Github ის რეპოზიტორია(რეპოზიტორიის ლინკი უნდა ატვირთოთ კლასრუმზე), რომელსაც დაუკავშირებთ Dagshub - ს, სადაც დალოგავთ თქვენს ექსპერიმენტებს.

რეპოზიტორიაზე უნდა იყოს შემდეგი ფაილები:
model_experiment.ipynb - ძირითადად იმუშავებთ აქ, ყველა ეტაპი და შესაბამისი დალოგვა უნდა იყოს ამ ფაილში. Cleaning, Feature Engineering, Feature Selection და Training ის ნაწილები ერთმანეთისაგან უნდა იყოს გამოყოფილი შესაბამისი Heading - ით jupyter notebook - ში.
model_inference.ipynb - ამ ფაილში უნდა ხდებოდეს test set ზე პროგნოზი, საუკეთესო მოდელის გამოყენებით უნდა დააგენერიროთ submissions, რომელსაც ატვირთავთ kaggle - ზე და შეაფასებთ თქვენს მოდელს. ეს მოდელი უნდა იყოს შენახული Model Registry - ში და იქიდან უნდა დაალოუდოთ პირდაპირ და ისე გამოიძახოთ predict.
READM[README.md](README.md)E.md - ქართულ ენაზე დეტალურად უნდა აღწეროთ ყველა ეტაპზე რა მიდგომები გატესტეთ და თითოეულს რა შედეგი ჰქონდა.
თუ თქვენთვის უფრო მოსახერხებელი იქნება model_experiment შეგიძლიათ რამდენიმე notebook ში გადაანაწილოთ და ყველაფერზე ერთ ფაილში არ იმუშაოთ.
